64-Bit Addition Calculator

SiliconJackets February – March 2026

Full RTL-to-GDSII implementation of a 64-bit addition calculator with FSM-controlled SRAM interfacing. This project covered the complete digital design flow: RTL design, class-based functional verification, synthesis, place-and-route, and static timing analysis.

Details
  • Wrote 64-bit adder RTL in SystemVerilog using a cascaded 32-bit ripple carry adder, a result buffer register, and a finite state machine (FSM) controlling sequential 32-bit SRAM read and write operations.
  • Developed a class-based SystemVerilog verification testbench with constrained random stimulus, a golden model scoreboard for result checking, and SystemVerilog Assertions (SVA) for FSM state behavior, achieving 99% code coverage.
  • Executed the full RTL-to-GDSII physical design flow using Cadence Genus for logic synthesis and Cadence Innovus for floorplanning, placement, routing, and sign-off. Performed static timing analysis and resolved all design rule check (DRC) violations.
