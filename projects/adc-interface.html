ADC Firmware Interface

HyTech Racing January 2026

Developed a reusable C++ firmware interface for MAX1146-9 series ADCs used across HyTech Racing's PCBs. The interface improved measurement precision from 12-bit to 14-bit resolution and enabled accurate battery current monitoring for coulomb counting.

Details
  • Designed a templated C++ interface class supporting both single-ended and differential output modes across all 8 channels, compatible with any model of the MAX1146/1147/1148/1149 ADC family over SPI, allowing for code reuse across multiple PCB revisions.
  • Analyzed PCB schematics and MAX1146-9 component datasheets to derive and apply correct scaling factors and offsets for battery current measurement via a shunt resistor, enabling coulomb counting and reducing 10–20% error in battery voltage estimation compared to the previous 12-bit ADC implementation.
