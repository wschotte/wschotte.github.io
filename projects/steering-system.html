Steering System / Interface

HyTech Racing February – April 2026

Designed and implemented firmware for a digital steering sensor interface and overlying steering system on HyTech Racing's electric formula car. The project upgraded the car to a dual-sensor steering system by adding a 14-bit digital rotary encoder alongside the existing analog sensor, improving precision, noise immunity, and enabling safety-critical arbitration logic for software-driven torque vectoring and state estimation.

Details
  • Selected UART with an RS422 transceiver over SPI for the RLS Orbis rotary encoder interface after identifying noise interference risk from proximity to front electric motors near the steering column. This traded some sampling speed for significantly improved noise immunity.
  • Developed a C++ firmware interface for the Orbis encoder on a Teensy 4.1, including bit masks, command constants, an error struct, and public functions for data sampling and sensor calibration, all designed to integrate cleanly into HyTech's large cooperative-scheduler codebase.
  • Built a steering system layer on top of the interface, implementing a rolling buffer of readings, min/max tracking, timestamped data, wheel/steering angle conversions, and sensor arbitration logic with error flagging based on velocity, sudden changes, and out-of-range values.
  • Implemented software calibration triggered through a multi-board CAN message chain: button press on VCF → packed into CAN message → received by VCR FSM → state validation → calibration command sent back to VCF → steering system recalibrates parameters.
  • Debugged hardware issues using a logic analyzer and oscilloscope. Found a damaged RS422 transceiver causing low voltage levels on the line, confirmed firmware output was correct, and resolved the issue by replacing the transceiver.
  • Validated the system on-car using serial debug prints and Foxglove to cross-check CAN data received by the Drivebrain (Raspberry Pi), confirming smooth, low-latency steering angle data with correct error flag triggering across calibration edge cases.
